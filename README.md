# Тестирование Интернет-магазина с использованием Selenium и PyTest

Этот проект содержит набор тестов для автоматизации процесса покупки в интернет-магазине с использованием Selenium и PyTest. 

Тесты охватывают авторизацию, добавление товара в корзину, оформление заказа и проверку корзины после покупки.

## Установка

1) Клонируйте репозиторий: git clone https://github.com/Rufatello/QA_Auto.git

2) Установите необходимые зависимости: pip install -r requirements.txt

### Запустите тесты:
1) Откройте терминал

2) Перейдите в директорию проекта, если вы еще не там

3) Запустите тесты с помощью команды: pytest
